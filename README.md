# edu-www

## Project setup
```
yarn install
```

## Setting Host

```
127.0.0.1 edu-www.aitschool.com
```

由于 Chrome http 会自动跳转 https ，在微信登录体系的开发中可以尝试用 firefox 或者其他浏览器。

> 测试微信登录需要开 80 端口和配置 host
